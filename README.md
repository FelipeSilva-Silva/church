# ChurchManager - Sistema de Gerenciamento para Igrejas
## Descrição
ChurchManager é um software projetado para atender as necessidades de pequenas igrejas no gerenciamento de seus recursos. O objetivo principal é fornecer uma ferramenta simples e prática para ajudar igrejas a organizarem seus membros, finanças e eventos, bem como manter o controle sobre as filiais. Esta primeira versão do software inclui funcionalidades básicas que serão expandidas conforme o desenvolvimento do projeto.

Funcionalidades Principais
1. Login
Sistema de autenticação de usuários.
Diferentes níveis de acesso para administradores e membros da igreja.
2. Gerenciamento de Membros
Cadastro de novos membros com dados pessoais, como nome, endereço, telefone e data de nascimento.
Atualização e exclusão de registros de membros.
Organização dos membros em diferentes categorias (ex.: visitantes, membros regulares, líderes).
Relatórios para visualização do crescimento da comunidade.
3. Gerenciamento Financeiro
Controle de entradas e saídas de dinheiro.
Registro de dízimos, ofertas e outras fontes de receita.
Registro de despesas com descrição, data e valor (ex.: contas da igreja, eventos, manutenção).
Geração de relatórios financeiros mensais e anuais para melhor visualização das finanças.
4. Calendário de Eventos
Criação e gerenciamento de eventos da igreja (cultos, reuniões, celebrações, etc.).
Integração com o calendário de membros para lembrar de datas importantes.
Notificações automáticas para lembrar os membros dos eventos futuros.
5. Gerenciamento de Filiais
Cadastro e gerenciamento de filiais da igreja matriz.

Controle centralizado de membros e finanças para todas as filiais cadastradas.
